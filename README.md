# Accident Insurance Model

Проєкт для моделі страхування від нещасних випадків.

## Опис

Цей проєкт реалізує модель страхування від нещасних випадків з можливістю розрахунку страхової премії та виплат.

## Структура проєкту

```
AccidentInsuranceModel/
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/example/insurance/
│   │   │       ├── App.java
│   │   │       └── AccidentInsuranceCalculator.java
│   │   └── resources/
│   └── test/
│       └── java/
│           └── com/example/insurance/
│               └── AccidentInsuranceCalculatorTest.java
├── build.gradle
├── settings.gradle
└── README.md
```
